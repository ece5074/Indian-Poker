# Indian-Poker

Description
----------
더 지니어스 게임 시즌1 에서 나온 게임 중 인상깊었던 인디언 포커를 프로그램으로 구현
기본적으로 Player vs CPU로 가능하나, TCP/IP Socket을 사용하여 Player vs Player의 대전형식도 가능하게 해놓았음.
